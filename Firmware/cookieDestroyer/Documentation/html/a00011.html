Detailed Description

Board specific joystick driver header for the rubberDuckling.

Board specific joystick driver header for the rubberDuckling (http://fletchtronics.net/bumble-b). The rubberDuckling board does not contain any joysticks, and this file is included due to the implementation of custom boards into the LUFA library.

Note
This file should not be included directly. It is automatically included as needed by the joystick driver dispatch header located in LUFA/Drivers/Board/Joystick.h.
